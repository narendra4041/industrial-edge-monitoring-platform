# Changelog

All notable changes to the Industrial Edge Monitoring Platform will be documented in this file.

This project follows a structured SDLC and uses Conventional Commits for meaningful change history.

Format examples:

* `feat`: new feature
* `fix`: bug fix
* `docs`: documentation change
* `test`: test change
* `refactor`: code restructuring
* `chore`: maintenance or tooling
* `ci`: CI/CD change
* `security`: security improvement
